# Sweet Treats - Bakery E-Commerce Website

A responsive bakery e-commerce website built with vanilla HTML5, CSS3, and JavaScript. Browse fresh, organic baked goods and manage your shopping cart seamlessly.

## Key Features
- *Responsive Product Catalog*: Browse and filter bakery items with a clean, mobile-friendly interface
- *Shopping Cart System*: Add/remove items, view totals, and proceed to checkout
- *User Authentication*: Simple login system for personalized shopping experience
